# 木泽OPC Roadmap

## Vision

利用AI打造属于自己的One Person Company。

一个人 + AI团队，提高个人生产力，创造数字资产。

---

# Phase 1：基础建设（0-30天）

目标：

建立AI创作与开发基础设施。

完成：

- Mac mini开发环境搭建
- Windows AI工作站配置
- GitHub项目建立
- Codex工作流建立
- 内容生产流程建立

输出：

- 第一个AI项目
- 第一批内容作品

---

# Phase 2：内容系统（30-90天）

目标：

建立稳定内容生产能力。

方向：

- AI学习记录
- Codex开发过程
- AI自动化实践
- OPC建设记录

完成：

- 发布30条内容
- 建立选题库
- 建立脚本模板
- 建立复盘系统

---

# Phase 3：AI自动化系统（90-180天）

目标：

让AI成为自己的工作团队。

建设：

- AI Agent
- 自动化工作流
- 内容生产自动化
- 个人知识库

输出：

- 可重复使用工具
- 自动化脚本

---

# Phase 4：产品化（180-365天）

目标：

让One Person Company产生价值。

探索：

- AI工具
- 数字产品
- 开源项目
- 服务模式

建立：

- 个人网站
- GitHub资产
- 用户社区

---

# 长期目标

成为一个由：

人 + AI

组成的新型个人公司。

用AI扩大个人能力边界。
